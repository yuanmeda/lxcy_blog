在 node 层一般可以使用 fetch、node-fetch、axios

2. fetch

- 默认不带 cookie
- 错误不会 reject
- 不支持超时设置

```js
// 可以写一个通用的异步超时函数 🔥
function handlerAsyncTimeout(asyncFn, timeout = 3000) {
  return new Promise((resolve, reject) => {
    asyncFn().then(resolve, reject);
    setTimeout(() => reject, timeout);
  });
}
```
