Axios 是一个基于 promise 的 HTTP 库，可以用在浏览器和 node.js 中

## 快速搭建一个服务

首先使用 [json-server](!https://github.com/typicode/json-server/) 快速搭建一个服务来配合使用

```js
// 1. 安装包
npm install -g json-server
// 2. 创建json 文件 db.json
// 3. 启动服务
json-server --watch db.json
```
